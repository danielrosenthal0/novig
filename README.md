# Weather Event Planner — Daniel Rosenthal (Novig Take-Home)

A Vite + TypeScript SPA that helps you decide whether you can host your event outside — enter a location and date range, and get a weather breakdown with a go/no-go recommendation.
## Public Deploymnet
[Try the app here](https://novig-five.vercel.app/)
## Stack
- **Weather data:** Visual Crossing API
- **Location search:** Geoapify
- **Charts:** Chart.js
- **Styling:** Tailwind CSS

## Setup
```bash
npm install
npm run dev
```
> Add API keys to a `.env` file for Visual Crossing and Geoapify
